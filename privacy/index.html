Privacy Policy – RapidStryde Download Manager

Effective date: [2025-09-18]

Overview

This extension runs locally in your browser to help start downloads. It does not send any data to the developer or third-party servers. No analytics, advertising, or tracking is used.

Data the extension accesses to function

  • Page and media URLs – used to determine what to download.
  • HTTP request headers (e.g., User-Agent, Accept-Language, and related sec-* headers) captured for media requests so the download can be reproduced.
  • Site cookies for the sites you download from, only to authorize the same media request that your browser could make.

This information is used only when you click the extension button/context-menu or when a download is intercepted. It is kept in memory and not persisted by the extension.

Where data goes

  • Data is sent to a local helper on your own machine at 127.0.0.1 to perform the download. It is not transmitted to the developer.
  • The helper then requests the media from the original website using the necessary headers/cookies to authenticate the request.

What is not collected

  • No personal information, account details, or browsing history is collected.
  • No data is sold, shared, or used for advertising or analytics.
  • No data is stored in cloud services; the extension does not have remote servers.

Permissions explained

  • contextMenus – add a “Download with YT4K” right-click item.
  • cookies – read cookies for the specific site you’re downloading from to authorize the request.
  • tabs – read the current tab’s URL/title for naming and routing downloads.
  • webRequest – observe media request headers so downloads can be replicated.
  • downloads / downloads.shelf – intercept/hand-off downloads to the local helper.
  • navigation / webNavigation - To make user-initiated downloads work, we call chrome.webNavigation.getAllFrames({tabId}). There is no subscription to navigation events or any logging / data transmitting to anywhere.
  • <all_urls> – enable the overlay on sites where media appears.

Third-party services

The UI may load a font stylesheet from Google Fonts for appearance only; no user data is sent to us.

Data retention

The extension does not persist the accessed data; it is used ephemerally for the download action.

Your choices

You can disable or remove the extension at any time. If you use the optional local helper, refer to its documentation for any additional logs or settings.
